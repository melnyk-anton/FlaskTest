Виникла помилка

Список доступних url_адрес:

  • /api/register (для реєстрації)
  • /api/delete (для видалення)
  • /api/excel (для роботи з ексель файлом)
  • /api/change/int:id (для редагування студента, int:id - його id в бд)
  • /api/student/all (поверне всіх студентів, які зареєстровані, у файлі json)
  • /api/student/int:id (поверне студента у файлі json за його id в БД, int:id - його id в бд)
